# KTB4-Justin-Week1

Java CLI 기반 RPG 프로젝트입니다.
메이플스토리를 참고하여 직업 선택, 사냥, 레벨업, 전직, 무기 구매 등의 기능을 구현했습니다.

---

## 프로젝트 소개

사용자는 게임 시작 시 직업군과 직업을 선택할 수 있으며,
사냥을 통해 경험치와 메소를 획득하고 레벨업 및 전직을 진행할 수 있습니다.

---

## 주요 기능

* 직업군 및 직업 선택

    * 전사
    * 마법사
    * 궁수
    * 도적
    * 해적

* 사냥 시스템

    * 레벨 구간별 몬스터 등장
    * 몬스터 처치 시 경험치 및 메소 획득

* 레벨업 시스템

    * 경험치 획득 시 자동 레벨업
    * 최대 레벨 50

* 전직 시스템

    * 10 / 20 / 30 / 40 레벨 전직
    * 전직 차수에 따른 경험치 및 메소 배율 증가
    * 전직 차수에 따른 스킬 변화

* 무기 구매 시스템

    * 직업군별 무기 제한
    * 메소 사용

* 상태 확인

    * 직업
    * 레벨
    * 경험치
    * 메소
    * 장착 무기
